# SavingsGoalApp

An Android app that helps you **set, track, and achieve savings goals** 


## Architecture & Tech Stack

This project uses:
- Kotlin for Android development.
- MVVM architecture for separation of concerns and testability.

## Requirements

Before you build/run the app:

- Android Studio (latest stable)
- Minimum SDK version as defined in `build.gradle`
- A real device or emulator with Android API 26+ (or your configured minSdk)
